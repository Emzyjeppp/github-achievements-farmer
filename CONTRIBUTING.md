# Contributing to GitAchieve - GitHub Achievement Hunter Guide

Thank you for your interest in contributing to GitAchieve! This is a bilingual, glassmorphic single-page dashboard designed to help developers guide and automate their GitHub achievements collection using rate-limit protected templates.

## How to Contribute

1. **Fork the Repository:** Create a personal copy of this repository on your GitHub account.
2. **Create a Feature Branch:** Create a branch for your UI improvements or new badge templates (e.g., `git checkout -b feature/new-badge-guidelines`).
3. **Make Changes:** Keep style modifications in `styles.css` adhering to our clean dark glassmorphism system and logic updates in `app.js`.
4. **Submit a Pull Request:** Open a PR back to the main branch of this repository.

## Design Guidelines
- Keep panels and cards rounded to `16px` and interactive elements/buttons rounded to `30px` pill-shapes.
- Use `:active` scaling at `0.97` for tactile push feedback effects.
- Keep the design clean, flat, and free of overused solid backgrounds or glowing border anti-patterns.
